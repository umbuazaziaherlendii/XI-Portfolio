Tentang Saya

Halo! Saya Umbu Azazia Herlendi, lahir pada 18 September 2008 di Perdana. Saat ini saya duduk di kelas XI jurusan Pengembangan Perangkat Lunak & Gim (PPLG) di SMK Wikrama Bogor.

Saya passionate di dunia game development, khususnya sebagai programmer menggunakan C# (terutama dengan Unity) dan sebagai artist 2D/3D. Saya suka menciptakan pengalaman interaktif yang menyenangkan, dari coding logic game hingga mendesain aset visual yang menarik.

Di sekolah, saya belajar dasar-dasar programming dan desain, tapi hobi saya lebih ke game—membuat prototype sederhana seperti platformer atau puzzle game. Saya terus belajar untuk menjadi developer game profesional di masa depan!
